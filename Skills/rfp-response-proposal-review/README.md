# RFP Response - Proposal Review

Audits the generated proposal draft across three dimensions — strategy fidelity, RFP coverage, and content sourcing — and produces a Review Report Word document before any human reviewer opens the draft.

## What you get

- Review Report Word document with overall status: READY FOR HUMAN REVIEW or REQUIRES REVISION
- Strategy fidelity findings — ToC structure, theme coverage, tone, drafting-note adherence
- Draft Coverage Map — every RFP row classified as Covered, Placeholdered, Thin, or Missing
- Content sourcing confidence ratings per section — flags potential hallucinations
- Placeholder Register — every placeholder with owner and priority
- Prioritised action list for the pursuit team
